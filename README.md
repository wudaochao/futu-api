# futu-api

基于富途 OpenAPI 的程序化交易学习与实践项目。

## 项目目标

本项目计划通过富途 API 逐步搭建一套程序化交易基础框架，覆盖行情获取、账户查询、交易执行、风控控制与策略验证等能力。

## 当前阶段：第一阶段

第一阶段的核心目标是系统学习富途官网 OpenAPI，并跑通基础能力，为后续程序化交易框架打基础。

### 学习重点

- 理解富途 OpenAPI、OpenD、SDK 之间的关系
- 完成开发环境搭建与 OpenD 连接验证
- 学习行情接口：股票列表、实时行情、K 线、逐笔、摆盘等
- 学习交易接口：账户信息、持仓、订单、下单、撤单等
- 熟悉回调机制、错误码、权限限制与接口频率限制
- 区分模拟交易与真实交易的使用边界

### 建议实践顺序

1. 安装并启动富途 OpenD
2. 安装 Python SDK 并完成连接测试
3. 查询基础市场与股票信息
4. 获取历史 K 线与实时行情
5. 订阅行情并处理回调数据
6. 查询模拟账户资产与持仓
7. 使用模拟账户完成下单、撤单、订单查询流程
8. 整理 API 使用注意事项与常见错误

## 后续阶段规划

### 第二阶段：基础交易框架

- 封装行情模块
- 封装交易模块
- 增加统一日志与异常处理
- 增加基础风控规则
- 支持模拟交易执行流程

### 第三阶段：策略开发与验证

- 实现简单策略模板

- 完成模拟盘策略验证
